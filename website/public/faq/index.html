Frequently Asked Questions

How to configure Hyper Gwent?

By default, Hyper Gwent is only enabled for the Gwent subreddit. But you can enable it for any website.

After you installed Hyper Gwent on your browser, you can access its status and options from the extension icon in your browser menu.

How do I enable Hyper Gwent on a website?

You can tell if the extension is running on any page by checking the extension icon. When the icon is green, then the extension is running, otherwise the icon is gray.

To enable (or disable) Hyper Gwent on the whole website you are visiting, open the extension menu. Click Enable on this site (or Disable) to tell Hyper Gwent whether it should run across the current website. For example, if you are on https://reddit.com/r/gwent and click Enable on this site, then Hyper Gwent will be enabled for all pages from reddit.com.

You can be more precise, and click Enable/Disable on this page only. This will enable or disable Hyper Gwent for the current page URL, and all pages that start with the same URL. Can it work anywhere?

In theory, Hyper Gwent can work anywhere. It does its best to work on complex apps like twitch.tv. But it can happen that Hyper Gwent does not play well with a particular website. If you have issues interacting with a website after enabling Hyper Gwent, disable the extension and tell me about it.

Is it safe?

Yes, the extension is safe. But this is a legitimate concern. Firstly, you should know that the extension source code is public. You can know everything about it, and you could even build the extension yourself and install it manually on your computer. Secondly, the extension is verified by Chrome and Firefox teams, before it is published on their store.

You may also be worried by the permissions requested by the extensions. On installation, your browser will warn you about what the extension will have access to. I did my best to limit the required permissions to the minimum. Here is an explanation for each one:

  • "The extension can read and change all your data on the websites you visit."
    This is because you should be able to enable the extension on any website, and to find cards in the page you visit, it needs to read the page's content.
  • "The extension can read your browsing history."
    The extension must know the website you are currently visiting, so that it can be enabled on the websites of your choice, and so it can activates when your current tab is one of those websites. As a result, the extension knows the websites you visit. In other words, it could read your browsing history.
