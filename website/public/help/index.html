Help
This is the help page for the browser extension Hyper Gwent

Configuring Hyper Gwent

By default, Hyper Gwent is only enabled for the Gwent subreddit. But you can enable it for any website.

After you installed Hyper Gwent on your browser, you can access its status and options from the extension icon in your browser menu.

Enable Hyper Gwent on any website

You can tell if the extension is running on any page by checking the extension icon. When the icon is green, then the extension is running, otherwise the icon is gray.

To enable (or disable) Hyper Gwent on the whole website you are visiting, open the extension menu. Click Enable on this site (or Disable) to tell Hyper Gwent whether it should run across the current website. For example, if you are on https://reddit.com/r/gwent and click Enable on this site, then Hyper Gwent will be enabled for all pages from reddit.com.

You can be more precise, and click Enable/Disable on this page only. This will enable or disable Hyper Gwent for the current page URL, and all pages that start with the same URL. Limitations

Hyper Gwent is not meant to be enabled on every website. It does its best to work on apps like messenger.com without breaking them. But Hyper Gwent might not play well with some websites. If you have issues interacting with a website after enabling Hyper Gwent, disable the extension and tell me about it.
